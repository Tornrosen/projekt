Om webbplatsen

På många sätt handlar livet om att inspirera och inspireras för mig. Jag älskar historier.
När jag inte läser eller lyssnar på musik eller tittar på någonting, tycker jag om att skapa själv. Framför allt att skriva.
Idén till den här webbplatsen kom därifrån, och jag hoppas att den är lite inspirerande i alla fall. Jag vill åtminstone gärna skriva i ord och se vad som kommer upp!
Jag har använt tre olika API:er för att skapa innehållet.
WordsAPI för synonymer: Länk.
WordCloudAPI för att skapa ordmoln: Länk.
Slutligen, Pexels API för bilder: Länk. Hovra över fotografierna för att se fotografnamnen.
Länk till automatiskt genererad JSdoc-dokumentation: Klicka här.
